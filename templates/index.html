Frequently Asked Questions

How do I find the Workshop ID?

Go to the Steam Workshop item page and copy the ID from the URL. It's the long number after ?id=

Is this tool completely free?

Yes! This service is completely free to use. Donations are optional but greatly appreciated.

Where do my downloads go?

All downloaded files are saved to our server and can be accessed from the Downloads page.

Can I download multiple items?

Yes, just enter each Workshop ID one at a time. You can track all downloads in the status list.

What file formats are supported?

We support all common Workshop item types including maps, mods, skins, and other content.

How long do downloads take?

Download times vary based on file size and server load. Most items download in under a minute.
